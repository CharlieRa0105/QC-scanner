## Decisions made
- Autonomous execution with per-task verify-command checkpoints; stop at hardware.
- ROS graph built to interface, not to running MoveIt/hardware (Ra's call).
- Real-arm cartesian tracing stays behind `QC_ALLOW_SCAN_TRACE` (default OFF).
- Reachability = the controller's IK, not a software sphere.
- "Go to point" + the viewer's debug frame use the controller base frame so
  commanded vs reported can be validated 1:1.
